User Manual

What is the User Manual?

This manual helps users understand how to use all major modules of the Security Management System. Follow the instructions below to perform common tasks.

How to Add an Employee

  1. Open Employee Management from the sidebar.
  2. Click Add New Employee.
  3. Enter Employee ID.
  4. Enter Employee Name.
  5. Select Agency.
  6. Select Designation.
  7. Click Submit.

How to Add Leave Record

  1. Open Leave Management from the sidebar.
  2. Click Add New Leave.
  3. Enter Employee ID.
  4. Select Leave Type (REST, ABSENT, EL, CL, ML).
  5. Choose From Date.
  6. Choose To Date.
  7. Click Submit.

How to Mark Attendance

  1. Open Attendance from the sidebar.
  2. Search for employee if required.
  3. Select attendance status for each employee.
  4. Click Save to record attendance.

How to Generate Reports

  1. Open Reports from the sidebar.
  2. Select the report date using the date picker.
  3. Review deployment details in the table.
  4. Click Print Duty Roster if you need a printed copy.

Important Note

Always verify employee information before saving records. Contact the administrator if any issue occurs.
